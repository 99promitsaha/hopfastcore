# HopFast Workspace

Project is now organized by app:

- `frontend/` -> React + Vite + Tailwind client app
- `backend/` -> Express + MongoDB API

## Run Frontend

```bash
cd frontend
npm install
cp .env.example .env
npm run dev
```

## Run Backend

```bash
cd backend
npm install
cp .env.example .env
npm run dev
```
